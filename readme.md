# eventeve 🎉

eventeve is a web-based event management system developed using PHP and MySQL. It allows users to create, manage, and organize events efficiently.

---

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 👥 Multi-user system (user-specific events)
- ➕ Add events
- ✏️ Edit events
- ❌ Delete events
- 🔍 Search functionality
- 📊 Dashboard with event count
- 🎨 Modern UI with gradient design & animations

---

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Server: XAMPP  

---

## ⚙️ How to Run

1. Install XAMPP  
2. Place project in `htdocs`  
3. Start Apache & MySQL  
4. Import database in phpMyAdmin  
5. Open: `http://localhost/eventeve`
